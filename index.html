    Welcome to Animatman Pages.

    this a jQuery Plugin for simple build animation just using sprite image, and set width/height. 這是一個JQuery的套件用來製作簡單的動畫， 只要使用Sprite的圖檔，然後設定寬和高即可

    Demo Page

    Example / 範例 :

    Demo2: 
    $.animatman($('#demo1'),
'./images/sprite_width_01.png',
'75',
'100', {
})
$.animatman($('#demo2'),
'./images/sprite_height_01.png',
'100',
'75', {
    fps: 12
})
$.animatman($('#demo3'),
'./images/ramonaflowers.png',
'125',
'180', {
    fps: 12
})
